# 정보처리기사

## 일정(정기 1회 기준)

- 필기 원서접수: 2/25(화) ~ 2/28(금)
- 필기시험: 3/22(일)
- 필기 결과발표: 4/3(금)
- 실기 원서접수: 4/6(월) ~ 4/9(목)
- 실기시험: 5/9(토) ~ 5/24(일)
- 최종 결과발표: 6/26(금)
