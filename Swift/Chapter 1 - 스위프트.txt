Chatper 1 - 스위프트

- iOS, macOS, watchOS, tvOS 등 대상으로 사용하기 위해 애플이 스위프트를 만듦
- 오픈소스 프로젝트
- 언어의 자유도가 높아서 장단점이 있음

특징 3가지
1. Safe
2. Fast
3. Expressive

다중 프로그래밍 패러다임 언어
1. 함수형 프로그래밍 패러다임
2. 프로토콜 지향 프로그래밍 패러다임
3. 명령형 프로그래밍 패러다임(절차지향)
4. 객체지향 프로그래밍 패러다임

함수형 프로그래밍 패러다임
- 프로그램의 상태 변화 없이 데이터 처리만 집중 -> 대규모 병렬처리가 쉬움
- 함수를 일급 객체로 다루기 때문에 함수 자체를 인자나 반환 값으로 사용할 수 있음

* 프로토콜 지향 프로그래밍 패러다임은 23장에서 다룸

- Xcode의 Playground 기능을 이용해서 프로젝트 생성없이 간단하게 테스트 가능
