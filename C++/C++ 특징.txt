C++

- 1979년 'C with classes'로 시작
- 1983년 비아네 스트로스트룹에 의해 C++ 개발
- C=C+1의 의미
- C 문법의 절차적 프로그래밍, 클래스로 대표되는 객체지향 프로그래밍, 템플릿으로 대표되는 제네릭 프로그래밍 삼위일체(멀티 패러다임)

- C++은 OOP를 지원
- 다른 객체지향 언어에서는 많은 부분을 런타임에서 처리하며 메모리를 자동으로 관리하지만, C++은 최대한 많은 것을 컴파일타임에 처리하는 것을 지향하고 메모리를 프로그래머가 직접 관리하기 때문에 클래스 디자인에서 상당한 차이를 보임
- C++11 이후(모던 C++)는 제네릭 패러다임이 상당한 영향을 끼침
- 기능이 다양하고 성능이 프로그래밍 언어 중 손에 꼽힐 정도로 좋지만, 양이 방대하고 배우기 어려움

- ISO(International Organization for Standardization, 국제표준화기구)에서 98년 최초 표준안 제정(C++98)
- C++03 이후 3년마다 개편 11, 14, 17 ... 20(개발중)
